Algorithms




ETF+CLD Algorithm




LOG Algorithm


ETF+CLD Algorithm extracts a set of coherent, smooth, and stylistic lines that effectively capture and convey important shapes in the image. Many parameters are involved in this algorithm which enable users to process images freely to a great extent.

But a high-contrast background will be filled with a dense set of lines although this area may be perceptually unimportant. FDoG filter operates on a local kernel and thus a global- scale subjective contour may be hard to detect.

LOG (Laplace of Gaussian filter) is a very simple and fast algorithm. It is the most recommended algorithm when generating cartoon style input image. It can clearly delineate the characters and objects in the image.



Prewitt Filter Algorithm




Canny Algorithm


Prewitt Filter is an alternative method when LOG produced a disappointing result for a realistic image input. Its performance will be better for these input.

But there is a kindly reminder: an input image from real world (e.g. photo) is usually not a good candidate for generating an image suitable for coloring. Please choose cartoon style or abstract style images.

Canny algorithm can produce coloring page with a narrow and pale outline. This is a alternative for user who like this style.
